# 🤖 AI Chatbot (Streamlit + LangChain)

## 🚀 Overview
A conversational AI chatbot built using Streamlit and LangChain, capable of maintaining chat history and generating intelligent responses using OpenAI.

## 🧠 Features
- Chat UI using Streamlit
- Conversational memory (chat history)
- Prompt engineering
- OpenAI integration via LangChain
- Environment-based API key management

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- OpenAI API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run src/hello.py
